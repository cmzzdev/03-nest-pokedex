# Run in dev ⚙️
1. Clone repository

2. run
```
yarn install
```

3. Nest CLI must be installed
```
npm i -g @nestjs/cli
```

4. Run mongo DB
```
docker-compose up-d
```

5. Load data by seed
```
http://localhost:3000/api/v2/seed
```

## App developed with 📋 
* Nest
* Mongo DB
* Docker
